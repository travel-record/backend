===== HTTP Request

include::{snippets}/google-login/http-request.adoc[]

===== Notes

- 이 API는 사용자가 구글 계정을 통해 로그인할 때 사용됩니다.
- 요청 바디에는 구글로부터 받은 인가 코드가 포함되어 있어야 합니다.

===== HTTP Response

include::{snippets}/google-login/http-response.adoc[]

===== Notes

- 응답에는 사용자 ID, 닉네임, 토큰 및 리프레시 토큰이 포함됩니다.
- 유효하지 않은 인가 코드로 로그인을 시도하면 오류가 발생합니다.

===== Potential Errors

- `INVALID_GOOGLE_AUTHORIZATION_CODE`: 잘못된 구글 인가 코드로 요청한 경우
